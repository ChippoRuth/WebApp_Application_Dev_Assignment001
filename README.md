# Assignment Details
This assignment 1 for Application Development class, aimed at creating a simple web map that allows users basic interactions and conduct simple computations client-side.
This assignment used OpenLayers V4 as the web map library. There were two tasks for this assignment , as below :
Task 1 : Adding a distance measurement button to the HTML file as well as an Event Listener to the JavaScript, for the button and implemeent the measurement functionality in the web map. 
Task 2: Getting User´s location by adding a button that when clicked by user, the map view should center on their location
